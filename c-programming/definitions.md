# Definitions

### Comments&#x20;

Comments in C start with // and go to the end of a line. Multiline comments begin with /\* and end with \*/  .&#x20;

### Separators&#x20;

Expressions in C are separated by semicolons (;). This tends to appear at the ends of lines. &#x20;

### Booleans&#x20;

Ignoring the bool type, zero is false, and non-zero is true.&#x20;
